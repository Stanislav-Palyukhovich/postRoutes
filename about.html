Description

This add-on is developed to be a time-saver during the route upload process to Track-POD.

Tips

• You can mix up the order of the Sample columns. However, changing column names is not allowed because they are the API Route object variables.
• The Sample contains an additional table which is linked with DriverName, DriverLogin, and DriverPassword columns by formulas. Add your drivers' credentials there and Excel will put them to the main table automatically.
• Please make sure, the data in your Excel file corresponds with the Drivers information on the Track-POD side.

Important

• Check the imported routes in Track-POD carefully after each import.
• Track-POD has no relation to this add-on.
